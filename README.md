# Waalaxy Test Technique

Installation:

```sh
git clone https://github.com/Charles-Chrismann/waalaxy.git
cd waalaxy
npm i
```

Build et démarrage de l'application:

```sh
npm run start
```

Exécution des tests:

```
npm run test
```

Développement:

```sh
npm run dev:api
```

```sh
npm run dev:frontend
```
